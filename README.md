# sku-level-inventory-forecasting-pivot_tab-power_query-
SKU-Level Inventory Demand Forecasting &amp; Replenishment Model (Excel)

## 📌 Project Overview
This project simulates a real-world supply chain scenario where inventory decisions are made at SKU-location level.

The goal is to:
- forecast product demand
- calculate safety stock
- determine reorder points (ROP)
- identify replenishment needs

## 📊 Dataset
The dataset contains 90 days of historical sales for multiple:
- SKUs
- warehouse locations

## ⚙️ Tools Used
- Microsoft Excel
- Pivot Tables
- Conditional Formatting

## 📈 Key Features
- SKU-location level demand forecasting
- Average Daily Sales calculation
- Safety Stock calculation
- Reorder Point (ROP)
- Replenishment status (ORDER / OK)

## 🧠 Business Value
This model helps:
- prevent stockouts
- optimize inventory levels
- support data-driven replenishment decisions
